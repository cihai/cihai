# Conversion

{mod}`cihai.conversion` translates between Python strings and legacy {term}`CJK` encodings. Reach
for functions such as {func}`cihai.conversion.python_to_ucn` and
{func}`cihai.conversion.ucn_to_unicode` when you need exact encoding conversion outside the
high-level {term}`UNIHAN` lookup flow.
