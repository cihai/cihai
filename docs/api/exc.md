# Exceptions

When using cihai via Python, catch project-specific failures through this module. All Cihai-specific
exceptions inherit from {exc}`~cihai.exc.CihaiException`. Configuration validation on
{class}`cihai.core.Cihai` raises {exc}`cihai.core.CihaiConfigError`.
